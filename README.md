# pie

## Introduction

Pie (abbr. Pie Is an Editor) is a cross-platform text editor, intended to be fast, efficient, easy-to-use, while having basic support for essential text editing features.

## Features

- graphical user interface

- modal editing

- syntax highlighting

- configurable
