# pie

## Default Modes

### Normal

- left: t
- right: n
- up: r
- down: h
- next word: w
- prev word: b
- line selection: x
- match: m
- delete selection: d
- replace selection: c
- undo: z
- redo: y

### Insert

- from normal to insert: i
- back to normal: esc
- backspace: backspace

##  

- open file: ctrl + o
- save file: ctrl + s
- save file as: ctrl + shift + s
- undo: ctrl + z
- redo: ctrl + y
